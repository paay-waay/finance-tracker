# AI / Codex Project Rules

This is a static Finance Tracker PWA.

## Do not change without explicit request

- Existing app settings
- Existing data model
- Existing UI layout and visual design
- Import/export behavior

## Allowed maintenance changes

- GitHub Pages compatibility
- PWA manifest and icon metadata
- Service worker cache/version updates
- README / CHANGELOG / deployment documentation

## Deployment assumptions

- Repository root is the published folder.
- GitHub Pages source is `main / root`.
- App should install on iPhone as `Finance Tracker`.
